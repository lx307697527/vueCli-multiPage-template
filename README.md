# 多页脚手架


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8082
npm run dev

# serve with hot reload at localhost:8082 进入联调func环境
export NODE_ENV=func
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report(打包时报告生成的包大小 )

npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
